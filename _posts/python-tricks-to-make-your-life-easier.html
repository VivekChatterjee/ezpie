Python tricks to make your life easier

Everyone knows about Python, especially those who have just written their own first code. But no doubt that, python is the most popular and very simple to understand. But like any other language, python also have many tricks which I guess non of us knew(till now!).

So here are the top 6 tricks that will help you code better rather than code more!

Trick 1: more like golang

If you have ever used golang then you may know about the smallest way of defining a variable in it, using the := operator.

This operator is known as the walrus The latest version of python now have this operator used for defining a variable in a conditions statement, loop etc. Just like other normal languages we know about.

NOTE: your python version must be above 3.8 for this trick

Trick 2: zip the code

Imagin if you want to generate a dictionary using two list, how will you do that? By using the zip() function!

This function allows us to merge two list into one, cool way to combine things together.

Output:
Bill Gates
Larry page
Elon mask

How about merging these two list into one dictionary?

Output:

{"Bill":"Gates", "Larry":"page", "Elon":"mask"}

Trick 3: Remove the replica

One list can have more than one replica of an item in it, therefore there are chances that you may not get the results you what sometimes(Basically always) so to prevent this what you can do is store that data using the set() function(set data type)

Output:

{64, 67, 36, 70, 40, 60}

Trick 4: safety first

Passwords can be hacked, so to prevent that you can use the getpass() function from the getpass library.

Output:

password:
You login was successful

The fanny part is that, when you will type you wont get to know the password, as it does not shows!

Trick 5: From the lest to the highest

Try this, you have given a list of numbers like this [25,62,6,2,53,63,56]. Now try to return a list that contains these numbers in ascending order.

To do this the easy way just use the sorted() function. This is a list method used to return a list in a ascending order.

Output:

[2,6,25,53,56,62,63]
